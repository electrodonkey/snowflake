# Abstract
This project is a small wireless Christmas decoration that can be coupled in a network of snowflakes. It is based on the TI MSPM0L1306 microcontroller and the ST S2-LP sub-GHz transceiver.

# Inspiration 
- [Kingpin DEFCON 27 badge](http://www.grandideastudio.com/wp-content/uploads/dc27_bdg_schematic.pdf)

  - 24 PIN RGE Package (VQFN)

  - 24 PIN RGE Package (VQFN)
